## ğŸ“‹ Screenshot Guidelines

### **Technical Requirements**
- **Format**: PNG (for transparency support)
- **Quality**: High resolution (at least 1200px wide)
- **Browser**: Use Chrome/Firefox for consistent rendering
- **Zoom**: 100% browser zoom for accurate sizing

### **Composition Tips**
- **Center the content** in the screenshot
- **Include enough context** but avoid too much empty space
- **Ensure text is readable** at smaller sizes
- **Capture hover states** where relevant (buttons, links)
- **Show interactive elements** (open dropdowns, modals)

### **File Naming Convention**
```
XX-descriptive-name.png

Examples:
01-homepage-hero.png
02-events-grid.png
03-submission-form.png
```

## ğŸ”§ Taking Screenshots

### **Method 1: Browser Developer Tools**
1. Open DevTools (F12)
2. Use device toolbar (Ctrl+Shift+M)
3. Set to "Responsive" or "Desktop"
4. Take full page screenshots

### **Method 2: Browser Extensions**
- **Chrome**: "Full Page Screen Capture"
- **Firefox**: Built-in screenshot tool (Shift+F2)

### **Method 3: System Tools**
- **Linux**: `gnome-screenshot` or `flameshot`
- **Mac**: Cmd+Shift+4
- **Windows**: Snipping Tool or Win+Shift+S
